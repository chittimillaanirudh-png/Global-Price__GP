# 🌍 About Global Price

Global Price (GP) is a revolutionary metric system for monetary value, designed to establish one standard clear of sovereign frictions. Just like meters measure distance and kilograms measure weight, the GP standard measures intrinsic global value independent of local currencies or pricing anomalies.

The platform combines:
- 📏 Baseline Valuation Mapping
- 💵 Real-Time Currency Independent Calculation
- ⚓ GFRB Physical Anchor Basket Normalization
- 📉 VAT, Tariffs, and Margin Friction Stripping
- 📊 195 Countries Coverage
- 🔄 Daily Live Data Sync (World Bank & Frankfurter Indexes)

into a single interconnected ecosystem.

# 💡 The Engine Pipeline

Our four-step calculation sequence powers the entire standard:

### 1. 💵 Enter Price
Provide the local market currency value of any standard product.

### 2. ✂ Strip Frictions
Remove regional VAT taxes, local import tariffs, margins, and logistics scores.

### 3. ⚙ Compute GP
Normalize against the physical basket and apply PPP/CPI inflation correction indexes.

### 4. 🌍 Get 195 Prices
Project predicted purchase cost across 195 nations in native currencies instantly.
